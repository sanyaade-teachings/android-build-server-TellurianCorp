Setup & Configure from Local IDE

1. Run the installer from your IDE machine (via SSH):

ssh user@server 'cd /var/www/html/android-build && ./install.sh'

Replace user@server with your SSH credentials. The installer will prompt you for configuration and start the web server.

2. Configure device address from your IDE machine using curl:

curl -X POST http://server-ip:8000/api/device \ -H "Content-Type: application/json" \ -d '{"address": "192.168.1.20:5555"}'

Replace server-ip with your server's IP address, and 192.168.1.20:5555 with your actual device IP address and port.
